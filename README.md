# verge-test

# install

```shell
npm install
```

# Run Cypress test from terminal
```sh
npx cypress run --browser chrome
```

# Run Cypress test from UI

## 1. Open Cypress UI
```sh
npx cypress open
```

## 2. Choose "E2E Testing"

## 3. Click "Start E2E Testing in Chrome"

## 4. Open "Specs"

## 5. Click "verge.cy.js" to run it
